Este repositorio contiene actividades realizadas como tarea entregable para la asignatura de Ecoinformática, 
en el Máster en Conservación, Gestión y Restauración de la Biodiversidad de la Universidad de Granada.

Autor: Carlos Guzón García

El repositorio es también un proyecto de RStudio y tiene activado control de versiones con git. Archivos:

- mi_script.R --> Script donde se realizó el control de versiones de la Actividad 2.
- tendencia_invernantes_Donana.rmd --> Documento RMarkdown con control de versiones.
- tendencia_invernantes_Donana.pdf --> PDF final renderizado a partir del anterior.

Carpetas:
- data: contiene la serie temporal de datos de censos terrestres descargada del ICTS.
- docs: contiene el documento de metodología de censos terrestres, publicado en 2014.
- images: contiene las 2 imágenes renderizadas dentro del RMarkdown.
